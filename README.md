# NodeJS Boilerplate

Setup for NodeJS REST API applications.

Includes:

- NodeJS & Express Config
- Auth Middleware
- Error Middleware
- Routes Setup
- Express Validator
- DB Connection Config
- Nodemon

## Installation

1.  `git clone`
2.  `npm install`
3.  `npm run dev`
4.  `to connect to db populate .env with variables`
